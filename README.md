## Why?

Services like Releasebot track official announcements. We track actual changes — including ones that never appear in any changelog.

|  | Releasebot | Claude Patch Notes |
|---|---|---|
| Official release notes | Yes | Yes |
| Silent documentation changes | No | Yes |
| Line-by-line diffs | No | Yes |
| Breaking change detection | No | Yes |
| Free & open source | No ($59/mo) | Yes (MIT) |

## Features

- **Silent Change Detection** — Flag changes that don't appear in release notes
- **Breaking Change Alerts** — Auto-detect deprecated APIs and migrations
- **Line-by-Line Diffs** — See exactly what changed
- **AI Summaries** — Claude Haiku generates concise change summaries
- **Calendar View** — Browse changes by date
- **Search** — Find changes by keyword
- **RSS & JSON Feeds** — Subscribe in your reader
- **Webhooks** — Discord & Slack notifications
